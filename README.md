# Airbnb Homepage Clone

This project is a front-end clone of the Airbnb homepage created using HTML and CSS.  
The purpose of this project is to practice layout design, styling, and responsive web development fundamentals.

## Project Overview
The project replicates the visual structure and layout of the Airbnb homepage, focusing on clean UI, card-based layouts, and consistent spacing.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid

## Key Features
- Sticky navigation bar
- Property listing cards using CSS Grid
- Clean and modern layout inspired by Airbnb
- Organized and reusable CSS structure

## What I Learned
- Structuring large HTML documents
- Using Flexbox for alignment and spacing
- Creating grid-based layouts with CSS Grid
- Managing project structure for better readability
